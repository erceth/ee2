JSFlags is a robot AI tank game. The tanks' movements, game engine, collision detection, controls and AI logic are all written in Node.js and Javascript. The view portion uses HTML canvas manipulated using Javascript.

The tanks take commands from a Node server via Socket.io. A programmer takes the dummy AI code and improves it to create their own awesome AI and then battle against other programmer's AI.

When I worked at Unicity the IT department held a JSFlags tournament to see who built the best AI script.

I created a video where I explain how I upgraded the dummy AI to be able to navigate around obstacles and avoid other tanks. https://www.youtube.com/watch?v=omqTgNCXfbQ

How to create a game:

  • From the game server page select “Create Game.”
  • Select one of the game options.
  • Select “Create game” to spin up a new JSFlags node instances based on the selected options.
  • Select “Join” to join the game you just created and the game will be opened in a new tab.

How to join a game:

  • From the game server page select “Join.”

How to quit a game:

  • Close the browser tab. Note: When no players are in a game the game self destructs.

How to control:

  • To select a tank click on a single tank or click and drag across multiple tanks.
  • While tank(s) is selected right click on location to command the tank to move.
  • Press any key while tank is selected to command it to fire.

Objective:

  • Score the most amount of points.
  • Points are earned by capturing other players flag and returning it to your base.
